Bill Splitter App

This was a personal project to practice using JavaScript.
I built this bill splitter web app that allows the user to split multiple bills evenly or customize the split. At the end of it, when the user decides to settle the bill, it outputs the directions on which users pay the other users and how much so that everyone pays the correct amount. I used a linked list data structure where each node contains the username, total amount paid and total amount consumed. I also used local storage to store the data so that the browser can be closed and refreshed without losing the data.
